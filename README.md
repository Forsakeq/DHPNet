# DHPNet

The implementation and resource release for **DHP**.

> This repository provides the pretrained model, required pretrained backbone, and preprocessed auxiliary files for inference / evaluation / reproduction.


## 📌 Overview

This repository contains the implementation and resources for **DHP**, including:

- **Pretrained model weights**
- **Pretrained Swin-Tiny backbone**
- **Preprocessed EDGE files**
- **Preprocessed GT files**

If you use this repository in your research, please cite our work.

---

## 📂 Released Files

The following files are provided for reproduction:

| File | Description |
|------|-------------|
| `DHP_best.pth` | The best pretrained model checkpoint |
| `swin_tiny_patch4_window7_224.pth` | Pretrained Swin-Tiny backbone weights |
| `pre_EDGE` | Preprocessed EDGE files |
| `pre_GT` | Preprocessed GT files |
